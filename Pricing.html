Pricin

I offer a couple of services; the first being a website building service, the second a website modernisation service and the third a maintenance service. For all I require a wireframe/sketch sheet of your requirements/website necessities. I also work under the agile approach to programming, so you will get regular updates from me with the code and the aestethic appeal of the site. All my websites also follow the Ethos and Ethics of your company closely. Once the website is up and running the maintenance service I offer keeps your website up to date with the latest browsers and devices. My pricing structure is as follows:

Number of pages
1 2 3 4-5 5-7 7+
Website Creation £50 £75 £100 £125 £200 £300+
Website Modernisation £35 £55 £70 £90 £140 £210+
Website Maintenance [per month] £30 £60 £80 £100 £120 £150+
Creation: The prices above include the following features: A navigation bar, text, tables, graphics implementation (graphics provided by customer), headers, footers, fonts available freely i.e. under Google fonts, backgrounds and CSS for all. Any additional features will be subject to discussion with myself in the project sketch sheet and through correspondance. Hosting is also not included in these fees, nor is any domain, this will also require further discussion.
Modernisation: The prices above include only the CSS for the features above. Any HTML editing will require an additional cost of £10 per page ontop of the price in the table above.
Maintenance: The fee for this is per month; it can be paid in advance for a set number of months or paid monthly. It includes any updates the website may need to keep up to date with new versions of browsers and devices, or if a new HTML standard is released. All my projects are in HTML 5, which is the current standard. If any additional features are included in the website (features not listed above) a supplimentary maintenance fee may be included, i.e. if PHP is used for a database in the site.
